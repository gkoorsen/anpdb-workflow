# MD Analysis Results

Curated MD analysis outputs copied from the generated `output/` workspace for sharing through Git.

Included result sets:

- `md_publication_cyp1b1_maob/`: publication-style CYP1B1 and MAO-B replicate figures and summary tables.
- `md_sglt2_rep1_analysis/`: validated SGLT2 rep1 figures and tables.

These folders contain compact figures and CSV summaries only. Raw trajectories, checkpoints, logs, and full MD run directories remain excluded from Git.
